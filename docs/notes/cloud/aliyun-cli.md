# Aliyun/AlibabaCloud CLI

## Install

- Install CLI
Check the [doc](https://github.com/aliyun/aliyun-cli#installation).

- Check available regions:

```
./aliyun ecs DescribeRegions
```

- Configure

```
./aliyun configure
```
