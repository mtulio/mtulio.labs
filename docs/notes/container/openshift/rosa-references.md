## Usage

### Common commands:

```bash
./rosa whoami
./rosa verify quota
./rosa create cluster --sts
./rosa create account-roles
./rosa create cluster --sts
./rosa create operator-roles --cluster mrbrosa
./rosa create oidc-provider --cluster mrbrosa
./rosa describe cluster -c mrbrosa
./rosa get nodes
./rosa describe cluster -c mrbrosa
./rosa list -c mrbrosa
./rosa list machinepools -c mrbrosa
./rosa list clusters
```
